# 🚀 Novae Rog: Hackathon Command Cheat Sheet

This document contains the exact `/goal` commands you can copy-paste into **Antigravity** the moment you arrive at the hackathon to launch parallel, automated build sprints.

---

## 🧭 The Core Principle: Specialized Goal Execution

Do **NOT** run a `/goal` command on `base-concept.md`. 

*   **Why?** `base-concept.md` is our high-level conceptual, business, and storytelling document designed for pitching judges and explaining the vision to human collaborators. It is descriptive rather than actionable code-spec.
*   **The Strategy:** The other three documents (`system-spec.md`, `multi-agent-architecture.md`, and `hackathon-demo-gui-spec.md`) contain the precise, granular developer requirements derived from the base concept. Feeding these specialized specs directly into `/goal` ensures laser-focused, bug-free, parallelized execution by my spawned subagents.

---

## 🛠️ The 3-Step Hackathon Launch Commands

Copy-paste these exact commands into your Antigravity chat to trigger the parallel build tracks:

### 1. Track 1: Smart Contracts (The GOAT L2 Rails)
> `/goal Read system-spec.md and build, compile, and deploy the Syndicate Vault, Merchant Stake Vault, and Escrow smart contracts on the GOAT L2 Testnet using Foundry.`

### 2. Track 2: The Air-Gapped Agents (The ClawUp Backend)
> `/goal Read multi-agent-architecture.md and build the four OpenClaw agent runtimes (ROSA, XCPA, CVAA, VRSA) on ClawUp.org, including the Prisma state-sync database backend.`

### 3. Track 3: The Front-End Dashboard (The Graphical GUI)
> `/goal Read hackathon-demo-gui-spec.md and front-end-spec.md to build the reactive Air-Gap Pipeline GUI dashboard in React/Tailwind, ensuring all data density overlays and Forest Council animations are fully implemented.`
